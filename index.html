Pemodelan 3D Kawasan Bumi Perkemahan Karang Pramuka Kaliurang

Integrasi Drone LiDAR dan SLAM-Based Mapping

Lihat Model 3D

LATAR BELAKANG

Bumi Perkemahan Karang Pramuka Kaliurang adalah area perkemahan yang berada di lereng Gunung Merapi, tepatnya di kawasan wisata Kaliurang, Sleman, Yogyakarta. Tempat ini menawarkan suasana alam yang sejuk dan asri dengan pepohonan pinus yang rindang, cocok untuk kegiatan kepramukaan, outbound, pelatihan, maupun rekreasi alam. Bahkan saat ini terdapat penyewaan skuter dan atv yang dapat digunakan pengunjung untuk berkeliling area perkemahan dan menikmati pemandangan sekitar dengan lebih santai. Fasilitas dasar seperti area camping, aula terbuka, toilet, serta akses air bersih tersedia untuk mendukung kenyamanan pengunjung. Potensi wisata alam yang ada di Kaliurang ini cukup tinggi sehingga diperlukan visualisasi 3D yang akurat untuk mendukung perencanaan dan pengembangan kawasan. UAV LiDAR mampu menyediakan data spasial terbaru namun memiliki keterbatasan dalam menangkap detail vertikal bangunan. Untuk mengatasinya, teknologi SLAM digunakan sebagai pelengkap karena mampu memetakan area kompleks secara real-time dan detail, termasuk lokasi yang sulit dijangkau. Integrasi data UAV LiDAR dan SLAM menghasilkan point cloud yang lebih lengkap dan bersih, sehingga siap digunakan untuk analisis dan perencanaan kawasan secara lebih optimal.

Peralatan yang Digunakan

FJD Trion P1

FJD Trion P1 adalah pemindai LiDAR portabel berbasis teknologi SLAM yang mampu melakukan pemetaan 3D tanpa membutuhkan sinyal GNSS. Dengan akurasi sekitar ±2 cm dan bobot yang ringan, alat ini cocok untuk pemindaian area indoor maupun outdoor secara cepat dan efisien.

Baca Selengkapnya

SG7 Geomate GNSS

SG7 Geomate adalah receiver GNSS geodetik dengan teknologi RTK dan sensor IMU untuk meningkatkan stabilitas pengukuran. Perangkat ini mampu melacak berbagai konstelasi satelit dan memberikan hasil survei presisi tinggi untuk kebutuhan pemetaan, konstruksi, dan pekerjaan geospasial lainnya.

Baca Selengkapnya

DJI Zenmuse L3

DJI Zenmuse L3 adalah sensor LiDAR udara yang mengintegrasikan laser jarak jauh, kamera 100 MP, dan sistem navigasi presisi dalam satu modul. Cocok dipasang pada drone Matrice seri 300/350, alat ini mampu menghasilkan point cloud berkualitas tinggi dengan jangkauan luas dan penetrasi vegetasi yang baik.

Baca Selengkapnya

Software yang Digunakan

FJD Trion

FJD Trion Scan adalah aplikasi pendamping untuk perangkat LiDAR FJDynamics yang berfungsi untuk memantau pemindaian, menampilkan point cloud real-time, dan mengelola data hasil pemindaian. Sedangkan FJD Trion Model adalah aplikasi desktop untuk memproses data LiDAR dari perangkat FJD Trion.

Baca Selengkapnya

DJI Terra

Perangkat lunak pemetaan yang dirancang oleh DJI untuk memproses data dari drone guna membuat model 3D dan rekonstruksi 2D. Perangkat lunak ini memanfaatkan teknologi inti fotogrametri dan LiDAR untuk menghasilkan data geospasial yang akurat

Baca Selengkapnya

Agisoft Metashape

Agisoft Metashape adalah software fotogrametri profesional untuk menghasilkan orthophoto, DEM, dan model 3D berkualitas tinggi dari foto udara atau darat.

Baca Selengkapnya

Global Mapper

Perangkat lunak sistem informasi geografis (GIS) yang dikembangkan oleh Blue Marble Geographics untuk sistem operasi Windows. Perangkat lunak ini menyediakan berbagai alat untuk mengelola, menganalisis, dan memvisualisasikan data geospasial.

Baca Selengkapnya

Cloud Compare

Perangkat lunak bebas dan sumber terbuka untuk memproses dan mengedit awan titik (point cloud) 3D dan jaring segitiga (mesh segitiga), yang awalnya dirancang untuk membandingkan dua awan titik 3D yang padat.

Baca Selengkapnya

Maptek

Maptek PointStudio adalah software pemrosesan point cloud industri untuk pertambangan dan rekayasa, digunakan untuk analisis geoteknik, volumetrik, dan kontrol deformasi.

Baca Selengkapnya

Sketchup

Perangkat lunak pemodelan 3D yang intuitif dan serbaguna untuk membuat, memodifikasi, dan berbagi desain 3D. Software ini banyak digunakan di berbagai bidang seperti arsitektur, desain interior, dan desain produk karena antarmukanya yang mudah dipelajari.

Baca Selengkapnya

CesiumJs

Pustaka JavaScript open-source yang digunakan untuk membuat visualisasi peta dan bola dunia 3D di web, serta membangun aplikasi berbasis geospasial 3D. Pustaka ini memungkinkan pengembang untuk memvisualisasikan data 3D seperti citra satelit, rupa bumi, dan Ubin 3D (3D Tiles).

Akuisisi Data GNSS

Akuisisi GNSS dilakukan untuk membuat dua titik Bench Mark (BM) permanen sebagai referensi akurat dalam pengukuran LiDAR. Pengukuran menggunakan GNSS geodetik dengan metode statik selama 2 jam menggunakan SG7 Geomate GNSS.

Akuisi Data Drone Lidar

Akuisisi drone LiDAR dirancang dengan jalur terbang otomatis dari utara ke selatan, ketinggian 100 meter, dan jarak antar lintasan 5 meter untuk cakupan luas dan overlap optimal. Drone dilengkapi sensor LiDAR, GNSS RTK, dan IMU, menghasilkan point cloud rapat serta data foto untuk orthophoto

Akuisi Data Slam Lidar

Akuisisi SLAM LiDAR menggunakan FJD Trion P1 difokuskan pada area detail LOD3 seperti bumi perkemahan dan villa. Perangkat dipasang GPS NTRIP untuk koordinat real-time, dikendalikan via FJD Trion Scan, dan dibawa melingkar (loop closure) dengan sudut 45°.

Pembuatan Orthophoto

Pengolahan orthophoto drone dimulai dengan koreksi orientasi dan posisi citra menggunakan metode Post-Processed Kinematic (PPK) dengan perangkat lunak Red Toolbox. Setelah koreksi, foto diolah menjadi orthomosaic menggunakan software Agisoft, menghasilkan orthophoto resolusi tinggi yang bergeoreferensi dan digunakan sebagai dasar dalam digitasi footprint bangunan dan penyusunan basemap.

Klasifikasi Point Cloud Drone Lidar

Point cloud drone LiDAR diolah menggunakan DJI Terra dengan mengimpor data LiDAR beserta koordinat base station. DJI Terra melakukan proses georeferensi otomatis dan menghasilkan klasifikasi awal antara ground dan non-ground. Klasifikasi ini mempermudah pemisahan permukaan tanah dari objek seperti bangunan dan vegetasi. Output berupa file LAS yang siap digunakan dalam analisis lanjutan dan pembuatan model elevasi..

Klasifikasi Point Cloud Slam Lidar

Data SLAM yang awalnya berada dalam format .fjdslam dikonversi menjadi LAS menggunakan perangkat lunak FJD Trion. Setelah point cloud terkonversi, klasifikasi dilakukan di Global Mapper untuk memisahkan titik ground dan non-ground. Karena karakteristik SLAM yang sensitif terhadap perubahan elevasi, koreksi manual diperlukan untuk memperbaiki kesalahan klasifikasi pada area curam atau objek rapat agar ground dapat teridentifikasi dengan benar.

Pembuatan Digital Terrain Model

DTM dibuat dengan mengekstraksi titik-titik ground dari seluruh point cloud yang telah diklasifikasi. Data ground tersebut diinterpolasi untuk menghasilkan permukaan tanah yang benar-benar mewakili topografi asli tanpa pengaruh pohon atau bangunan.

Pembuatan Basemap

Orthophoto dan DTM digabungkan di SketchUp dengan terlebih dahulu memasukkan kontur dari DTM kemudian menyesuaikan posisi orthophoto hingga tepat berada di permukaan tersebut. Orthophoto kemudian dijadikan tekstur pada permukaan DTM sehingga terbentuk basemap 3D yang realistis. Hasil basemap ini digunakan sebagai pondasi visual bagi pemodelan bangunan dan lingkungan sekitar.

Pembuatan Model 3D LOD2

Model LoD 2 diawali dengan digitasi footprint bangunan dari orthophoto menggunakan QGIS. Setiap footprint diberi atribut ketinggian yang diambil dari point cloud drone. Footprint tersebut kemudian diekstrusi di SketchUp untuk membentuk bangunan LoD 1, sebelum akhirnya ditambahkan bentuk atap dan detail struktur dasar sehingga menjadi LoD 2. Model ini menggambarkan bangunan dengan bentuk geometri dasar namun belum detail fasad.

Pembuatan Model 3D LOD3

Model LoD 3 dibuat untuk area yang memiliki point cloud SLAM beresolusi tinggi. Data point cloud SLAM dan drone diintegrasikan kemudian ditampilkan di SketchUp menggunakan plugin Undet. Dengan memanfaatkan kerapatan titik yang sangat tinggi, geometri bangunan dapat direkonstruksi secara akurat hingga mencakup dinding, jendela, pintu, struktur atap, dan ornamen kecil lainnya. LoD 3 menghasilkan model dengan detail visual mendekati kondisi nyata.

Cek AKurasi

Validasi dilakukan dengan membandingkan 13 dimensi objek pada model 3D dengan pengukuran langsung menggunakan pita ukur. Selisih antara dimensi model dan lapangan dihitung untuk memperoleh nilai RMSE, yang menghasilkan akurasi sebesar 0,0516 meter. Nilai ini menunjukkan bahwa model 3D memiliki ketelitian tinggi dan merepresentasikan kondisi lapangan dengan sangat baik.
